# 💎 Emerald Frontend

O **Emerald** é uma plataforma de gestão financeira pessoal e controle de teto de gastos. Este repositório contém o ecossistema do cliente (Frontend), que está sendo ativamente desenvolvido com uma interface de alta fidelidade, estruturada e com um design minimalista moderno .
---

## 🚀 Tecnologias Utilizadas

O ecossistema visual do Emerald foi construído utilizando as seguintes tecnologias:

* **[React.js](https://react.dev/)** — Biblioteca SPA para construção de interfaces componentizadas.
* **[Vite](https://vite.dev/)** — Build tool ultra-rápido para o ambiente de desenvolvimento.
* **[Tailwind CSS](https://tailwindcss.com/)** — Framework utilitário de CSS para estilização premium e responsiva.
* **[Lucide React](https://lucide.dev/)** — Pacote de ícones minimalistas e vetoriais.
